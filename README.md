# Sift: The Zero-Latency Interface for Human Intention

Sift is a surgical, high-speed, local-first todo system built for Chromium-based browsers. It is designed to be a "Secondary Pre-Frontal Cortex," offloading the burden of remembering so the brain can focus on processing.

## Project Structure

- `/app`: Next.js web application (The Dashboard).
- `/extension`: Manifest V3 Chromium Extension (The Courier).
- `/docs`: Core logic, vision, and strategy (Source of Truth).

## Quick Start for AI Agents

1. **Tech Stack:** Next.js (App Router), Tailwind CSS, Dexie.js (IndexedDB), Lucide Icons.
2. **Design System:** Surgical Light Mode (#FFFFFF / #111111 / #FF4F00).
3. **Primary Rule:** Keyboard-first. Zero latency. No shadows. No rounded corners.
