# Product Strategy & Value Proposition

**Value Proposition:** The Zero-Latency Interface for Human Intention.
**Audience:** "Thought Architects" & Keyboard-First Pros (Devs, Researchers, Founders).

## Core Principles (The "Sift Way")

### 1. Flow Preservation
Capture via `Double-Shift` or `Cmd+I` without leaving the active browser tab. Entry cost must be lower than the value of the idea.

### 2. Enforced Intentionality
"Decide Now, or Let it Go." Sift rejects the "Infinite Backlog." If it isn't scheduled with a Working Date, it doesn't belong in the long-term system.

### 3. Surgical Neutrality
Visual silence (Light Mode, 0px radius) to prevent eye fatigue. The UI should feel like a professional instrument (e.g., Leica, Teenage Engineering).

### 4. Local Sovereignty
Local-first architecture (Dexie.js). Your hardware is the source of truth; the cloud is a secondary mirror. Zero loading spinners.

### 5. Intent/Deadline Decoupling
Distinct handling of `workingDate` (when I plan to do it) vs. `dueDate` (when it is externally required).
